# Civil Rights Video Generator

## Installation

To install the required packages, run:

```bash
pip install -r requirements.txt
```

## Usage

To run the application, use the following command:

```bash
python main.py --output-dir output --fps 30 --resolution 1920x1080
```

## Customization

You can customize the functionality by modifying the Python scripts directly instead of using a GUI.

## Troubleshooting

If you encounter issues, here are some common Python-specific solutions:
- Ensure you have all dependencies installed via `pip`. 
- Check the console for Python errors and stack traces to identify issues.
