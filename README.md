# Civil Rights Video Generator

## Features
- Create impactful videos focused on civil rights themes.
- Easy-to-use interface for video customization.
- Import images and audio clips from external sources.
- Share videos directly on social media platforms.
- Supports various video resolutions and formats.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/coreybashan274-ui/civil-rights-video-generator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd civil-rights-video-generator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

## Usage Examples
- To create a new video:
  - Open the application, click on "Create New Video".
  - Follow the prompts to add your content.

- To customize your video:
  - Select a theme from the available options.
  - Upload your images and audio clips.

## Customization Guide
- You can customize the video background by:
  - Navigating to the settings menu.
  - Choosing from built-in themes or upload your own.

- Audio customization can be done in the audio section where you can:
  - Adjust volume levels.
  - Trim audio clips.

## Troubleshooting Section
- **Issue:** Application fails to start.
  - **Solution:** Ensure all dependencies are installed correctly using `npm install`.

- **Issue:** Video does not export.
  - **Solution:** Check for any errors in the console log. Ensure file paths are correct.
